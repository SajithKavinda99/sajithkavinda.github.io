Lesson 1: Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard language for creating web pages.

In this lesson, you will learn about the basic structure of an HTML document and common tags.

Lesson 2: HTML Structure

The basic structure of an HTML document includes several elements like <html>, <head>, <body>, and <title>.

Example structure:

<html>
    <head></head>
    <body></body>
</html>

Lesson 3: HTML Elements

HTML elements are the basic building blocks of HTML. They consist of a start tag, content, and an end tag.

Examples: <div>, <h1>, <p>

Lesson 4: HTML Attributes

Attributes provide additional information about an element. They are always inside the opening tag.

<img src="image.jpg" alt="Image description">

Lesson 5: HTML Links

Links are created using the <a> tag. The href attribute specifies the destination.

<a href="https://www.example.com">Visit Example</a>

Lesson 6: HTML Forms

Forms are used to collect input from users. They are created using the <form> tag, along with input elements like <input> and <textarea>.

<form> ... </form>
